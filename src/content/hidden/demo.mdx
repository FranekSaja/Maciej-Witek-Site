# Dokumentacja

Cała treść strony powinna być edytowana w plikach `.mdx` (markdown), bez konieczności edytowania plików z kodem strony lub plików z konfiguracją CSS strony

&nbsp;

Klasy CSS dla powtarzających się elementów strony zostały globalnie nadane w pliku `global.css`. Tak samo kolory. Daje to możliwość szybkiego dodawania nowych, niestandardowych elementów strony z poziomu plików `.mdx`, bez konieczności pisania CSS. Zostały też wykorzystane komponenty `.astro`, takie jak `Icon`. Poniżej znajdują się przykłady użycia gotowych elementów ostylowanych zewnętrznie w `global.css`, oraz instrukcja. Całość treści tej strony znajduje się w pliku `content/hidden/demo.mdx` w katalogu z kodem strony.

## Instrukcja użycia :

Markdown to prosty język formatowania tekstu, który pozwala w łatwy sposób tworzyć nagłówki, linki i inne elementy strony bez konieczności pisania CSS lub HTML. Poniżej przedstawiono podstawowe zasady użycia elementów w plikach `.mdx`:

&nbsp;

&nbsp;

&nbsp;

&nbsp;

&nbsp;

1. **Tworzenie nagłówków**

   - **Nagłówek H1**  
     Aby utworzyć nagłówek pierwszego poziomu (H1), należy postawić znak `#` na początku linijki, a następnie wpisać tekst nagłówka.  
     **Przykład:**  
     ```markdown
     # To jest nagłówek H1
     ```

   - **Nagłówek H2**  
     Aby utworzyć nagłówek drugiego poziomu (H2), należy postawić dwa znaki `##` na początku linijki.  
     **Przykład:**  
     ```markdown
     ## To jest nagłówek H2
     ```

     &nbsp;

&nbsp;

&nbsp;

&nbsp;

&nbsp;

2. **Tworzenie linków**

   Aby zamienić fragment tekstu w link, należy:
   - Umieścić tekst w nawiasach kwadratowych `[ ]`
   - Bezpośrednio po nich podać adres strony w nawiasach okrągłych `( )`  

   **Przykład:**  
   ```markdown
   [YouTube](https://www.youtube.com)
   ```
   efekt : [YouTube](https://www.youtube.com)


&nbsp;

&nbsp;

&nbsp;

&nbsp;

&nbsp;

3. **Komponenty Astro**

   Komponenty Astro to javascriptowe, wcześniej przygotowane elementy kodu, aplikowalne do pliku `.mdx`, widoczne na stronie jako interaktywne elementy. Wszystkie dostępne komponenty Astro są opisane w tej dokumentacji. **Przykład komponentu i jego zapisu (w dowolnym pliku `.mdx`):**

   **Model zapisu:**

   ```markdown
     Import [nazwa funkcji tworzącej komponent] from '[ścieżka do pliku zawierającej kod Komponentu]'

     <[nazwa funkcji tworzącej komponent] [parametery komponentyu (opcjonalne). zapis : [nazwaParametru={[wartość nadana parametrowi]}] Komponent może mieć nieograniczoną ilość parametrów, pod warunkiem, że zostały nadane w kodzie komponentu]>
   ```
**Przykład:**

   ```markdown
    import Icon from '../../components/Icon.astro';


      <Icon name="mdi:github" bordered={true} />
   ```  

**Efekt na stronie:**

&nbsp;

<Icon name="mdi:github" bordered={true} />
